# 📝 Mini Projet JavaScript

## Description

Ce projet est une petite application réalisée en **HTML**, **CSS** et **JavaScript**. Il regroupe trois fonctionnalités permettant de manipuler le DOM et de gérer les événements en JavaScript.

## Fonctionnalités

### ✅ 1. To-Do List
- Ajouter une tâche.
- Marquer une tâche comme terminée.
- Supprimer une tâche.

### 🌙 2. Mode sombre
- Basculer entre le mode clair et le mode sombre grâce à un bouton.

### ❓ 3. Quiz interactif
- Afficher une question avec plusieurs réponses.
- Vérifier la réponse choisie.
- Afficher si la réponse est correcte ou incorrecte.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (DOM et événements)
